# JobKar - Job Portal

A modern, fast, and fully responsive job portal landing page built with Next.js.

## ✨ Features

- **Responsive Design:** Optimized for Desktop, Tablet, and Mobile devices.
- **Hero Section:** Dynamic search bar and interactive illustrations.
- **Job Listings:** Clean grid layout for browsing opportunities.
- **Feature Showcase:** Beautifully designed cards highlighting core services.
- **Testimonials:** "Voices of Experiences" section with smooth mobile scrolling.
- **FAQ & Contact:** Integrated forms and interactive accordions.
- **Modern UI:** Built with premium aesthetics, smooth gradients, and micro-animations.

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/)
- **Styling:** CSS3 (Vanilla)
- **Deployment:** Vercel / GitHub Pages
